## Course Overview

This comprehensive course teaches LangGraph fundamentals through 5 hands-on labs. Each lab includes:

## Labs

| Lab | Topic | Description |
|-----|-------|-------------|
| 1 | Environment Setup | Python venv, LangChain, LangGraph installation |
| 2 | State & Nodes | Creating graph nodes, managing state |
| 3 | Conditional Edges | Routing logic in graphs |
| 4 | Persistence | Checkpointing and memory |
| 5 | Production Patterns | Error handling, monitoring, deployment |
